Figure1.Rmd: R script for generating Figures 1D-G, S1. Output in the 'Figure1' and 'FigureSX' folder.

Figure2.Rmd: R script for generating Figures 2B-E. Output in the 'Figure2' folder. 

Figure3.nb: Mathematica Notebook for developing the statistical mechanics model and generating Figures 3A-C, S4, and S5. Output in the 'Figure3' and 'FigureSX' folder.  

Figure3.Rmd: R script for generating Figure 3D using data generated in modelExp.py. Output in the 'Figure3' folder. 

Figure4.Rmd: R script for generating Figures 4A-D. Output in the 'Figure4' folder.

SupplementalMaterial.Rmd: R script for generating supplemental figures. Output in the 'FigureSX' folder.

barcodeQuant_LacZSpacing.Rmd: R script for taking raw barcode counts from 'looping_architectures' folder to generate expression dataframe fLP3.loop_MOPS_glu_expression.txt in 'processed_data' folder.

countsToSumExpression.R: R script for taking raw barcode counts from 'alternative_architectures' folder to generate expression dataframe LacZ_variant_exp.txt in 'processed_data' folder.

modelExp.py: Python script for generating expression predicted by model in Figure 3D.
